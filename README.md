# Recipe-Finder
# 🍽️ Recipe Finder Application

## 📌 Project Overview

Recipe Finder is a simple and responsive web application that allows users to search for meals by name using **TheMealDB API**. The application displays meal images, category, cuisine, ingredients, and cooking instructions in an attractive user interface.

---

## 🚀 Features

* 🔍 Search recipes by meal name
* 🖼️ Display meal image
* 🍴 Show meal category
* 🌍 Display cuisine (area)
* 📋 View complete ingredients list
* 👨‍🍳 Read cooking instructions
* 📱 Responsive design
* ⚠️ Error handling for invalid searches
* ⏳ Loading indicator while fetching data

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* TheMealDB API

---

## 🌐 API Used

The application uses **TheMealDB API** to fetch recipe information.

API Endpoint:

https://www.themealdb.com/api.php

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Search for a meal (Example: Chicken, Pasta, Pizza, Cake).
5. Click **Search** to view recipes.
